ibtrg

Name

ibtrg -- trigger device (device)

Synopsis

#include <gpib/ib.h>

int ibtrg(int ud);

Description

ibtrg() sends a GET (group execute trigger) command byte to the device specified by the device descriptor ud.

Return value

The value of ibsta is returned.
